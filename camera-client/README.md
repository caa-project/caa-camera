camera-client
=============

# Usage

```
python camera.py \
  --server="${CAMERA_SERVER}" \
  --width="${WIDTH}" \
  --fps="${FPS}" \
  --height="${HEIGHT}" \
  --quality="${QUALITY}"
```

# Requirements

- [picamera](http://picamera.readthedocs.org/en/release-1.8/install2.html)
  ```
  $ sudo apt-get update
  $ sudo apt-get install python-picamera
  ```
